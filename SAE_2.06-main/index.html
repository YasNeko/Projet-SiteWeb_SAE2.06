Vous cherchez à devenir développeur ?

Les métiers de l’informatique regroupent tous les métiers qui permettent le traitement de l’information par les ordinateurs.
Cela regroupe donc tous les emplois qui permettent de créer des systèmes informatiques/logiciels/applications. Mais aussi tous les métiers liés à la maintenance, la formation ou la commercialisation dans le domaine informatique.
Selon vos préférences, vous pourrez faire le même métier dans divers secteurs, par exemple : l’armement, les ESN, le nucléaire ou encore le sport.
