  • Développeur Informatique

    Le développeur logiciel participe à la création de divers logiciels, donc il doit toujours apprendre. Il prend en charge la conception d’un programme depuis le cahier des charges, jusqu'au déploiement.

    Outils utilisés:

    • Python
    • Java
    • C

    Etudes:

    • niveau bac + 2: BTS Informatique
    • niveau bac + 3: BUT Informatique
    • niveau bac + 5: Master/Ingénieur Informatique

  • Ingénieur en programmation linguistique

    L’ingénieur en programmation linguistique conçoit et paramètre des logiciels de traitement et rédaction de texte automatique.
    Il doit maîtriser sa langue de travail et différents langage de programmation (Java, Python, C)

    Outils utilisés

    • Python
    • Java
    • C

    Etudes:

    • niveau bac + 5: Master/Ingénieur Informatique
